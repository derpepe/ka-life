# KA-Life — Karlsruhe in Zahlen

Wöchentliche Infografik-Kolumne für [ka-life.de](https://ka-life.de) mit Karlsruhe-Bezug.

Jeden Samstag erscheint eine neue interaktive Infografik zum wichtigsten Karlsruhe-Thema der Woche — mit Daten, Fakten und satirischen „Torten der Wahrheit".

## Tech Stack
- React + Vite + Tailwind CSS
- Playfair Display + Source Sans 3 + JetBrains Mono
- SVG-basierte Infografik-Komponenten

## Development
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
NODE_ENV=production node dist/index.cjs
```

## Social Media Grafiken
Jede Woche wird automatisch eine 1080x1080px Post-Grafik unter `client/public/social/` generiert, die für Instagram, LinkedIn etc. verwendet werden kann.
